# english-room

## サイト名

英語学習室

---

## 新しい記事の作り方

1. Obsidianで新しいMarkdownファイルを作成します。
2. ファイル名を `002.md`、`003.md` … のように3桁の番号へ変更します。
3. 作成したMarkdownを該当する年フォルダ（例：`2026`）へ入れます。
4. Markdownの先頭にフロントマターを書きます。
5. GitHub Desktopでコミット・プッシュします。

GitHub Pages上ではJekyllがMarkdownをHTMLへ変換し、トップページにもカードとして表示します。

---

## 記事テンプレート

```markdown
---
layout: record
number: "#002"
date: 2026-07-01
year: 2026
permalink: /2026/002/
---

## 今日の単語

## 今日の熟語

## 今日の構文

## 今日の記録
```

「今日の記録」は自由記述欄です。

例えば

- 今日読んだ教材
- 学習した範囲
- 難しかったこと
- 気付いたこと
- 感想
- 明日へのメモ

などを自由に記録します。

---

## 運用ルール

- 1記事 = 1Markdownファイル
- ファイル名は3桁番号 + `.md`
- 記事番号は3桁（001, 002, 003 …）
- 年ごとにフォルダを分ける
- トップページの記事カードはGitHub Pages上で自動生成する
