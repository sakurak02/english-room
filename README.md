# english-room

## サイト名

英語学習室

---

## 新しい記事の作り方

1. `templates/article` フォルダをコピーします。
2. コピーしたフォルダ名を `001`、`002`、`003` … のように3桁の番号へ変更します。
3. 作成したフォルダを該当する年フォルダ（例：`2026`）へ入れます。
4. `index.html` をVS Codeで開きます。
5. 次の3か所だけを書き換えます。

   - `#XXX` → `#001`
   - `yyyy-mm-dd` → `2026-06-30`
   - `<!-- ここに書く -->` の部分へ本文を入力

6. トップページ（`index.html`）へ記事カードを1件追加します。
7. GitHub Desktopでコミット・プッシュします。

---

## 記事テンプレート

```
#XXX

yyyy-mm-dd

今日の単語

今日の熟語

今日の構文

今日の記録
```

「今日の記録」は自由記述欄です。

例えば

- 今日読んだ教材
- 学習した範囲
- 難しかったこと
- 気付いたこと
- 感想
- 明日へのメモ

などを自由に記録します。

## 運用ルール

- 1記事 = 1フォルダ
- ファイル名は常に `index.html`
- 記事番号は3桁（001, 002, 003 …）
- 年ごとにフォルダを分ける
- トップページには記事カードを追加する
